BRST対称性

String Theory DemystifiedでもBRST量子化を扱うが，ここでもわかった感じがしないので， BRST quantization and string theory spectraを参照した。また，部分的にString Theory も参照した。

BRST対称性

非物理的な自由度を拘束するために，Faddeev-Popov行列式を経路積分に挿入し，これを詳細に計算すると以下の作用を得ることは前の章で説明した。 \begin{eqnarray} S_{\mathrm{polyakov}}&=&\frac{1}{4\pi\alpha^{\prime}}\int d^{2}z\partial X\bar{\partial}X\nonumber\\ S_{\mathrm{ghost}}&=&\frac{1}{2\pi}\int d^{2}z(b\bar{\partial}c+\bar{b}\partial\bar{c})\nonumber \end{eqnarray} 実は，$S_{\mathrm{polyakov}}+S_{\mathrm{ghost}}$は以下の変換の下で不変である。$\epsilon$をグラスマン数の微小変換パラメータ とすると， \begin{eqnarray} \delta_{B}X&=&-i\epsilon(c\partial+\bar{c}\bar{\partial})X\label{eq:bram:47a}\\ \delta_{B}b&=&i\epsilon(T_{X}+T_{\mathrm{ghost}})\label{eq:bram:47b}\\ \delta_{B}\bar{b}&=& i\epsilon(\bar{T}_{X}+\bar{T}_{\mathrm{ghost}})\label{eq:bram:47c}\\ \delta_{B}c&=&-i\epsilon c\partial c\label{eq:bram:47d}\\ \delta_{B}\bar{c}&=&-i\epsilon\bar{c}\bar{\partial}\bar{c}\label{eq:bram:47e} \end{eqnarray} である。$T_{X}$は弦のエネルギー-運動量テンソル，$T_{\mathrm{ghost}}$はゴーストのエネルギー-運動量テンソルである。 まず，$X$に関する部分のラグランジアン密度を$\mathscr{L}_{X}$とかくと \begin{equation} \mathscr{L}_{X}=\frac{1}{4\pi\alpha^{\prime}}\partial X\bar{\partial}X\nonumber \end{equation} である。$\delta_{B}$を施し，$\epsilon$の一次の項を残すと \begin{eqnarray} \delta\mathscr{L}_{X}/i\epsilon&=&\partial(\delta_{B}X)\bar{\partial}X+\partial X\bar{\partial}(\delta_{B}X)\nonumber\\ &=&\partial c\partial X\bar{\partial}X+c\partial^{2}X\bar{\partial}X+\partial\bar{c}\bar{\partial}X\bar{\partial}X+\bar{c}\partial\bar{\partial}X\bar{\partial}X\nonumber\\ &+&\partial X\bar{\partial}c\partial X+\partial Xc\bar{\partial}\partial X+\partial X\bar{\partial}\bar{c}\bar{\partial}X+\partial X\bar{c}\bar{\partial}^{2}X\nonumber\\ &=&\partial c\partial X\bar{\partial}X+c\partial^{2}X\bar{\partial}X+\bar{\partial}\bar{c}\partial X\bar{\partial}X+\bar{c}\partial X\bar{\partial}^{2}X\nonumber \end{eqnarray} となる。ここで，運動方程式$\bar{\partial}\partial X=0,\partial\bar{c}=0,\bar{\partial}c=0$を用いた。$\partial^{2},\bar{\partial}^{2}$の項は 部分積分すると \begin{eqnarray} c\partial^{2}X\bar{\partial}X&=&-\bar{\partial}c\partial^{2}XX+c\bar{\partial}\partial^{2}XX=0\nonumber\\ \bar{c}\partial X\bar{\partial}^{2}X&=&-\partial\bar{c}X\partial^{2}X+\bar{c}X\partial\bar{\partial}^{2}X=0\nonumber \end{eqnarray} と運動方程式を使えば$0$になる。よって， \begin{equation} \delta_{B}\mathscr{L}_{X}=-i\epsilon(\partial c\partial X\bar{\partial}X+\bar{\partial}\bar{c}\partial X\bar{\partial}X)\nonumber \end{equation} となる。ゴーストのラグランジアン密度を$\mathscr{L}_{\mathrm{ghost}}$とかくと \begin{equation} \delta_{B}\mathscr{L}_{\mathrm{ghost}}=i\epsilon(T_{X}+T_{\mathrm{ghost}})\bar{\partial}c+i\epsilon b\bar{\partial}(c\partial c)+\mathrm{c.c.}\nonumber \end{equation} 第二項の符号が$+$なのは$\delta_{B}$の定義から$-$が$2$回でて$+$になっている。 ここで，$T_{\mathrm{ghost}}=2b\partial c+\partial b+\mathrm{c.c.}$であることを思い出すと \begin{equation} \delta_{B}\mathscr{L}_{\mathrm{ghost}}=i\epsilon T_{X}\bar{\partial}c+i\epsilon(2b\partial c\bar{\partial}c+\partial bc\bar{\partial}c+b\bar{\partial}c\partial c +bc\bar{\partial}\partial c)+\mathrm{c.c.}\nonumber \end{equation} となる。最後の項を部分積分すると \begin{eqnarray} \delta_{B}\mathscr{L}_{\mathrm{ghost}}&=& i\epsilon T_{X}\bar{\partial}c+i\epsilon(2b\partial c\bar{\partial}c+\partial bc\bar{\partial}c+b\bar{\partial}c\partial c -\partial bc\bar{\partial}c-b\partial c\bar{\partial}c)+\mathrm{c.c}\nonumber\\ &=&i\epsilon T_{X}\bar{\partial}c+\mathrm{c.c.}\nonumber \end{eqnarray} 最後に \begin{equation} \partial c\bar{\partial}X=\bar{\partial}c\partial X\nonumber \end{equation} を使えば \begin{equation} \delta_{B}\mathscr{L}=-i\epsilon T_{X}\partial c+i\epsilon T_{X}\partial c+\mathrm{c.c.}=0\nonumber \end{equation} となり，$S_{\mathrm{polyakov}}+S_{\mathrm{ghost}}$が変換$\delta_{B}$の下で不変であることが確認できる。

この変換$\delta_{B}$は提案した研究者の名前をとってBRST変換と呼ばれる。Faddeev-Popov行列式を経路積分に挿入したことは，ゲージ自由度を固定したことを意味しているのだから，ゲージ不変性は なくなっている。Faddeev-Popov行列式を変形していくとゴーストが現れたわけだが，ゴーストの持つ作用と理論本来の作用を加えた作用に別の不変性，言い換えると対称性が現れたのである。 これこそが，量子力学的なゲージ対称性なのである。
